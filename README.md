A small personal portfolio website cloned/stubbed to match the structure of https://github.com/akshatjha01/PRODIGY_WD_04

- `style.css` - styles
- `script.js` - small interactivity (nav toggle, smooth scroll)

How to run:
Open `index.html` in your browser. For local development with live reload, use any static server, for example:

```bash
# Python 3
python3 -m http.server 8000
# then open http://localhost:8000
```
